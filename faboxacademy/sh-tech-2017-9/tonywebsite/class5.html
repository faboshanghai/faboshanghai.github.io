  • In this class, the most important question which I have solved is how to use carol draw4 to do a engraving with the laser cut.
  • First, design the picture you want to engrave in Rhino!
  • Second!This is very very important, separater line and plane, and place them into two layers!
  • Third, set each colour to each layer, which means one layer only match to one colour.
  • Forth, this is also very important!Not to save it, but to export.
  • Fifth, change to the carol draw4, import the file!
  • Sixth, find the layer in Carol draw to ensure whether the colour corresponded to the layer is right!
  • Finally, check whether the parameter is right! and then the engraving will start!
