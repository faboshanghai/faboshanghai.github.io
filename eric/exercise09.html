Electronic Production

Make the Fab(tiny)ISP in-circuit programmer form a per-made design using Fab Modules and the Roland SRM-20.Solder the components and program the bord.

Images

Get Fab ISP circuit design from website of fabacademy.

Through the Photoshop to modify the picture, so that can not cut the smaller part of the width.

The use of fabmodules cutting file conversion, select the device used SRM20, and set the cutting parameters.

1/64 milling cutter used to cut PCB plate is very tiny.

Set milling cutter x, y coordinates, and then set the machine's z coordinates, very carefully put the milling cutter down from the machine, and then carefully tightened.

To the machine cutting instructions, the machine began carving PCB board.

After cutting, I use a knife to get it down.

I use the back of the knife to trim the board and use alcohol to clean the surface.

And that's what it's like.

We need to find the components, I heating electric iron, and put its head clean.

Start the board welding according to the circuit diagram!.

Be accomplished, is like this!.
